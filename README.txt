##Tunisia Property Price Analyzer:

This project analyzes property prices in Tunisia using real estate listing data.
The goal is to explore price differences between cities and understand how property characteristics relate to price.

##What’s inside:

-Data loaded from a CSV file

-Basic data cleaning and verification

-Simple statistical analysis of prices

##Visualizations:
-comparing average property prices by city
-price range and distribution across different property categories
-Relationship between property size and price
etc..

##Tools used:

-Python

-pandas

-matplotlib

-seaborn

-Jupyter Notebook

##How to run:

-Open the notebook Tunisia Property Price Analyzer.ipynb

-Make sure pandas, matplotlib, seaborn, and jupyter are installed

-Run the cells in order

##Files:

-Tunisia Property Price Analyzer.ipynb – analysis notebook
-Property Prices in Tunisia.csv – dataset
